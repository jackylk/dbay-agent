Code Agent

Code Agent 是一类 AI 驱动的自动编程代理，能够理解需求、编写代码、调试和测试。

代表性产品包括 [[Devin]]（[[Cognition]] 开发）和 [[Cursor]]。

评测基准主要使用 [[SWE-bench]]。

+ Wiki/图谱/对话各自独立，入口清晰
+ 不挤占知识库详情页
- 切换 Wiki↔图谱 需要点菜单
- Wiki 和 KB 的关联不直观（哪个 KB 的 Wiki？）
