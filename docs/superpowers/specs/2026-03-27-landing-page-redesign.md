# Landing Page 重构设计文档

## 概述

重构 DBay 官网 Landing Page，从当前的信息密集型长页面改为精简、现代、愉悦的首页，配合导航菜单承载深度内容。

**目标受众**：技术人员（开发者/Agent 构建者）+ 非技术人员（业务管理者），两者同等覆盖。

**设计参考**：Databricks（左文右图、专业感）、Notion（干净留白、居中布局）、Memos（大标题、产品截图）。

**核心原则**：
- 首页短而愉悦，一屏半看完
- 深度内容通过导航菜单进入子页面
- 三层递进价值主张
- 白底 + 蓝色强调 + 大量留白 = 明亮干净 + 专业数据感
- 中文为主，英文后补

## 导航菜单子页面（首页外的内容）

首页不承载详细内容，以下内容通过导航菜单进入独立子页面：

- `/product/lakebase` — Lakebase 详情

- `/product/memory` — 记忆库详情
- `/product/datalake` — 数据湖详情

### 集成子页面
效率优先、代码示例为主的集成指南：

- `/integration/mcp` — MCP 集成（Claude Code / OpenClaw 配置示例）
- `/integration/skill` — Skill 集成（Claude Code Skill 配置）
- `/integration/pg` — PostgreSQL 协议（psql / ORM 连接示例）
- `/integration/rest` — REST API（curl / SDK 示例）
- `/docs/api` — API 文档

## 视觉设计规范

### 配色
- 主色：#0073e6（蓝色）
- 背景交替：#fff（白色）/ #f8f9fb（浅灰）
- 文字：#111（标题）/ #222（副标题）/ #555（正文）/ #888（辅助）
- 强调色：
  - 记忆服务：#ff9800 → #e65100（橙色渐变）
  - 知识服务：#4caf50 → #2e7d32（绿色渐变）
  - 数据湖：#7b1fa2 → #4a148c（紫色渐变）
  - Lakebase：#0073e6 → #005bb5（蓝色渐变）
  - 作业引擎：#ff6f00 → #e65100（橙色渐变）
- CTA 按钮：蓝色实心 #0073e6 / 灰色 #f0f0f0

### 排版
- 大标题：44px, font-weight 800
- 数字条：32px, font-weight 800, 蓝色
- Section 标题：24-26px, font-weight 700
- 正文：14px, line-height 1.7
- 导航链接：16px, font-weight 500

### SVG 图表风格
- 所有元素带圆角（rx 8-16）
- 渐变填充（linearGradient）
- 投影效果（feDropShadow）
- 背景椭圆光晕（低透明度渐变椭圆）
- 连线使用半透明颜色（opacity 0.5）
- 协议/标签使用胶囊形徽章（rx = height/2）

### 布局
- 内容区 padding：48px 水平
- 三层递进交替布局：左文右图 → 左图右文 → 左文右图
- 图文间距：40px gap
- Section 间：1px solid #eee 分割线 + 背景色交替

## 技术实现

### 文件改动范围
- `lakeon-console/src/views/landing/LandingView.vue` — 主要重构文件
- `lakeon-console/src/layouts/PublicLayout.vue` — 导航栏改造（下拉菜单）
- `lakeon-console/src/router/index.ts` — 新增子页面路由
- 新增产品/集成子页面组件

### 保留
- Vue 3 Composition API + TypeScript
- 自定义 CSS（不引入外部框架）
- 亮暗双模式支持（CSS 变量）
- 中英双语 i18n（中文优先实现）
- 现有的 Trial 功能（"立即试用"按钮调用 /trial API）
